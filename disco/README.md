## Discord Annoy

so you wanna annoy someone jabroni on discord, but you don't wanna go overboard?  Well, have we got the tool for you.

Discord Annoy!

### How to Install

1. install latest version of python as usual
2. pip install pywin32 and keyboard
3. make sure annoy.py and poc1.txt are in the same directory
4. open up a terminal in the directory and run the script ie. python3 annoy.py
5. sit back and let the gradual mayhem ensue...
